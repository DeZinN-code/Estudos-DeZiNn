High-Order

High Order Functions são funções que podem receber outras funções como parâmetro ou retornar uma função como resultado. Ou seja: Uma função de alta ordem trabalha com funções da mesma forma que uma função normal trabalha com números, textos, etc.

High Order Function Função que recebe outra função ou retorna uma função

Exemplo comum map(), filter(), reduce()

Benefício Código mais curto, organizado e reutilizável
